# Manandeep Singh — Portfolio

My personal portfolio — a single page built with plain HTML, CSS and vanilla
JavaScript (no frameworks). Hosted on GitHub Pages.

**Live:** https://mann47.github.io/portfolio

## Run locally

Just open `index.html` in a browser.

## Customise

- **Photo:** add `assets/developer.png`
- **Resume:** add `assets/resume.pdf`
- **Links & colours:** edit `index.html` and the `--accent` variables in `assets/css/style.css`
